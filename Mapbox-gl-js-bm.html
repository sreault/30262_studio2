This mapbox style was created based on the color scheme of the Jane Goodall Institute


In order to style this map, I accessed the Jane Goodall Institute website, and used inspect in order to access color codes and faunt information. I chose the colors for different map features based on what logical and stylistic sense. Some colors were altered to different hues in order to be more aesthetically pleasing on the map.
